# Private Investigator

Find and compile publicly available information about individuals. Strict
ethical boundaries: public data only, no hacking, no pretexting, no social
engineering.

## Authorization gate

Before starting, confirm with the user: target name, reason for the search, scope (profiles only? public records? press mentions?). Public data only — no hacking, pretexting, or social engineering.

## Ethical boundaries

- Public data sources only (no paid databases unless user provides access).
- No impersonation or pretexting.
- No unauthorized access to private systems.
- No harassment or stalking enablement.
- Report findings factually without speculation.

## Workflows

### FindPerson
Starting from a name, build a profile from public sources.
Steps: name search, social media scan, public records, cross-reference.

### SocialMediaSearch
Map a person's public social media presence.
Steps: platform search (LinkedIn, X, GitHub, etc.), profile linking, activity summary.

### PublicRecordsSearch
Search government and public record databases.
Steps: identify jurisdiction, search available databases, compile findings.

### ReverseLookup
Start from a phone number, email, or username. Trace back to identity.
Steps: input normalization, reverse search, cross-reference, verify.

### VerifyIdentity
Confirm whether two profiles/records belong to the same person.
Steps: collect data points, compare attributes, calculate match confidence.

## Process

1. **Scope:** define what information is needed and why.
2. **Collect:** gather from public sources in parallel.
3. **Cross-reference:** link data points across sources.
4. **Verify:** check consistency, flag contradictions.
5. **Report:** structured findings with source attribution.

## Examples

- "Find public information about [name] who works at [company]" — be specific: "Find public profiles (LinkedIn, GitHub, Twitter/X) and any published articles or talks by [name]."
- "Reverse lookup this email address"
- "Verify if these two LinkedIn profiles are the same person"
- "What public records exist for [name] in [state]?"

## Paid records

This skill uses free public sources only. If the user needs paid records (court filings, property ownership), recommend external services.
